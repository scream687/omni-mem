To build the #1 repository on GitHub, we are not building a better hard drive. We are building a **Prefrontal Cortex for AI**.

## 2. The Cognitive Architecture (The "Brain")
We are abandoning simple vector-search logging in favor of a dynamic, self-organizing mind.

*   **Meta-Cognitive Reflection:** After every session, Omni-Mem runs an asynchronous "sleep cycle." It reviews the agent's successes and failures, distills them into permanent *Wisdom Rules*, and permanently alters how the agent approaches future problems.
*   **Zero-Prompt Intuition:** Instead of forcing the agent to stop and call a `search_memory` tool, Omni-Mem uses predictive streams to inject exactly the right context into the agent's working memory *before* the agent even asks.
*   **Regret & Anti-Pattern Eradication:** Omni-Mem aggressively catalogs mistakes. If a UI design was rejected, or a bug was introduced, it forms an "Anti-Pattern Antibody," mathematically preventing the agent from walking down that same dead-end path again.
